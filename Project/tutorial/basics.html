Basics of perl

Practical Extraction and Reporting Language.

In this tutorial you are going to learn how to write basic programs in perl.As this tutorial is specifically for biology/bioinformatics students,most of the examples and practice questions are related to biology.

Let us start with a simple program !!!!

       #!usr/bin/perl 
       print "Hello! world";
       EXIT;

Type these lines or copy the code above in your favourite perl editor (like Notepad)and save it as first.pl
Make sure that you have saved it in .pl format. Now go to command prompt,and change current working directory to the one in which you saved your program i.e
suppose you have saved your first.pl in the directory "c:\examples" .. Now to change your command prompt to this directory,first open you command prompt window and type the following command
cd\ and now the following command
cd examples. Now the command prompt changes to
C:\examples\>
Now type perl first.pl
C:\examples\>perl first.pl

If everything goes correct you will see the following output
Hello! world

Now its time to see what the code above actually meant

#!usr/bin/perl
This line is known as shebang line. This line should be included on the top of every perl program.This line tells the parser(Parser is the engine which converts your perl program into machine readable language i.e 0's and 1's)that it is a perl program.

print
print is a function which prints the output to screen.(We can also redirect output to files.I shall talk about it later).

EXIT
EXIT command tells the parser that it is the end of the perl program.EXIT command is optional in any program but it is suggested

Now its time to have a closer look at print function!!

print is a function which returns the standard ouput to the screen.standard output is the output which is generated by your perl program. As print is a function it accepts arguments which are to be printed on to screen.
Ex: print "AGTCATCGGTACGCTGCCC";
You can send as many arguments you want to the print statement seperated by commas i.e
print "AGC","GTC","TCA";
This statement prints AGCGTCTCA to the screeen.
you can also send arguments to the print enclosed by brackets
print ("AGC","GTC","TCA");(Same as the above print statement)

Datatypes and variables

Any programmming language is designed to work on data efficiently. This data is stored into placeholders called variables. In C/C++ we have lots of datatypes like int,float,string,struct,pointers etc etc.C/C++ are called strictly typed languages.A strictly typed language wont allow variables which hold data of one type to hold data of another type. for ex in C/C++ if one say variable v1 is of integer type,you have to store only integers in that v1 variable.you cannot store strings or characters into that v1 variable.Further more in strictly typed languages we have to declare all the variables before the actual code starts.

Perl is a Loosely typed language.Perl has only 3 basic data types

  1. Scalars - denoted by a $ symbol
  2. Arrays -denoted by a @ symbol
  3. Hashes -denoted by a % symbol
There is also another type called typeglob but it is not considered as true datatype.

We shall discuss about data types in the next chapter.
