The most recent of perl is perl 5.10 .Currently the version that is widely used in production environments is
perl version 5.8.8

perl version 5.0 is released around early 1980's. The previous version of perl i.e perl 1,2,3,4 are now obselete.

Versions from perl 5 has support for OOPS[Object oriented programming].(Like c++/JAVA)

[Dont worry if you dont know about oops.we will discuss oops in detail afterwards]

If you want to install ActivePerl 5.8.8 for windows i suggest you to visit www.perl.com[OR]www.activestate.com

All Unix/Linux systems have perl already installed.You just have to go to terminal and type the following line.
Perl -V
This command will show you the current version of perl that is installed on your system.Windows users can also type this command from CMD(command prompt/DOS)window after installing the perl,to check whether installation was succesful or not.
